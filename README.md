# ✈️ ATC Incident Analyzer

AI 기반의 항공 관제 사고 시뮬레이션 및 데이터 시각화 도구입니다. 
Mapbox 위성 지도를 활용하여 실제 항공 경로와 긴급 상황을 시각적으로 분석합니다.

## 🚀 주요 기능
- **실시간 항공 경로 시각화**: Mapbox 위성 지도를 기반으로 한 항공기 궤적 표시
- **긴급 상황 알림**: 항공기 결함 및 관제 상황에 따른 실시간 이벤트 로그
- **반응형 대시보드**: 고도, 속도, 위치 데이터를 한눈에 확인하는 인터페이스

## 🛠 사용 기술
- **Frontend**: React.js
- **Map Engine**: Mapbox GL JS
- **Deployment**: Vercel
